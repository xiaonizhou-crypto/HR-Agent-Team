# Quickstart

This prototype is meant to be used in the smallest possible way.

## Step 1
Pick one role.

Example:
资深后端研发工程师（电商 / 搜索方向）

## Step 2
Run Resume Screener on an incoming resume.

Output should answer:
- continue or not
- why
- next step

## Step 3
If needed, run Candidate Coordinator.

Use it for only 3 actions:
- request more info
- schedule interview
- summarize candidate reply

## Step 4
Run Hiring Ops Tracker.

Record only:
- current stage
- owner
- next step
- status

## Step 5
Write back to Feishu.

Use the smallest fields only:
- Candidate Name
- Role
- Current Stage
- Owner
- Next Step
- Status
- Last Update
- Notes

## Current loop
Resume in → Resume Screener → Candidate Coordinator → Hiring Ops Tracker → Feishu update
